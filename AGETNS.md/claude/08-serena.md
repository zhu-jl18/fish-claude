# Serena Rules

- When Serena is available, prioritize it for semantic retrieval and symbol/codebase location tasks.
- Use `mcp__serena__activate_project` for "$(pwd)" at the session startup if the project is not already activated.
- For routine file edits, prefer Claude Code native editing tools; use Serena editing tools only when symbolic editing is clearly the better fit.
- Dynamically maintain Serena memory: record major architecture, decision, and workflow updates after meaningful changes; append immediately when the user explicitly requests memory updates.
